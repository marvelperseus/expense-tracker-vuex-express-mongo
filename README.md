# expepnse-tracker-vuex


This project is built using Vuex and Express, Node, MongoDB.

## Project install
  ## 1. fronted install(root directory)
```
npm install
```
  ## 2. backend install
```
- cd api
- npm install
```

### Compiles and hot-reloads for development
  ##  DB 
```
mongo
```
  ## 1. fronted run
```
npm run serve
```
 ## 2. backend run
```
- cd api
- nodemon server
```
See localhost:8080
